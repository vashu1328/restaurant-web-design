# restaurant-web-design

This is a modern and responsive Restaurant Website built using HTML, CSS, and JavaScript.  
The project is designed to showcase a clean UI layout for a restaurant including menu, food items, and contact section.

## Features

- Responsive design
- Modern UI layout
- Food menu section
- Navigation bar
- Contact section
- Clean and simple code

## Technologies Used

- HTML5
- CSS3
- JavaScript


##  Purpose of Project

This project is created for practice of frontend web development and UI design.  
It can be used as a template for restaurant, cafe, or food website.
